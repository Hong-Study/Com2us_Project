## 피드백
1. 싱글톤 패턴을 사용하지 않고, Action이나 Func 델리게이트 형식으로 전달받아서 사용해라.
    - 객체 간의 연동은 최대한 느슨하게 하는 것이 좋다.
2. 유지보수가 좋은 코드를 작성해라
3. 방어적인 코드를 작성해라
4. 좀비 객체를 조심해라
5. 데이터만 바꿔도 기능이 바뀌도록 구현하는 것이 가장 좋다.
